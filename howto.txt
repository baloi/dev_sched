> install specific gem version:
  $ gem install rails -v 1.2.6
> install rake task for automatically generating a gems file from your
  Rails gem dependencies. Rerun the rake task whenever you change your
  dependencies.
  $ curl https://gist.github.com/101101.txt > lib/tasks/heroku.rake
  $ rake heroku:gems
  $ git add lib/tasks/heroku.rake .gems 
  $ git commit lib/tasks/heroku.rake .gems -m 
    "Adding gem manifest for Heroku."
> Heroku
  gem install heroku
  heroku keys:add
  heroku create

> runing autotest

  $ autoest -rails

  ~/dev_barn, ~/barn, midnight-runner
